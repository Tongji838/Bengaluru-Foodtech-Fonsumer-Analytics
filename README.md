# Bengaluru-Foodtech-Fonsumer-Analytics
Analysis and visualization of online food ordering platform data, demographic profiling, and consumer sentiment in Bengaluru

## Project Overview
This project analyzes user behavior and satisfaction levels for an online food ordering platform in **Bengaluru, India**. By cross-referencing demographic data (age, occupation, income) with consumer feedback, this dashboard provides actionable insights into market penetration and service performance.

The analysis is based on a dataset of **388 unique consumer entries**.


## 📈 Key Performance Indicators (KPIs)
* **Average Age:** 24.63 years (Highlighting a youthful, tech-savvy user base).
* **Customer Sentiment:** **81.7% Positive** (317 Positive vs. 71 Negative responses).
* **Market Scale:** 388 Total Orders analyzed.
* **Average Family Size:** 3.28 members.

## 🔍 Core Insights
* **Target Audience:** The platform is highly popular among **Students** and **Post-Graduates**. 
* **Income Correlation:** A significant portion of the high-volume user base falls into the "No Income" (Student) or entry-level salary brackets, indicating a high demand for value-based food services.
* **Geographic Hotspots:** Order density and larger family sizes are heavily concentrated in **Central and South Bengaluru**, particularly near residential and educational hubs.
* **Gender Trends:** The data shows a balanced engagement across genders, with slight variations in income-level distributions for ordering frequency.

## 🛠️ Tech Stack & Tools
* **Data Source:** `Online Foods.csv` (388 entries).
* **Visualization:** Power BI
* **Analysis:** Demographic Profiling, Geospatial Mapping, and Sentiment Analysis.

## 📂 Repository Structure
* `Online Foods.csv`: The raw dataset used for the analysis.
* `Dashboard_Preview.jpg`: High-resolution screenshot of the final visualization.
* `README.md`: Project documentation and insights.

## 🚀 How to Replicate
1. Clone this repository.
2. Open the `.csv` file in your preferred BI tool.
3. Map the `latitude` and `longitude` columns to a map visual to replicate the Bengaluru density heat map.
4. Filter by `Feedback` to identify specific demographic pain points.
